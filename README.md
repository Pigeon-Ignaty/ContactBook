# ContactBook
Приложение представляет собой адресную книгу, предназначенную для хранения, редактирования и управления списком контактов с сохранением данных в XML-файл.

## Описание классов

### MainWindow
Главный класс приложения, отвечающий за отображение интерфейса, взаимодействие пользователя с контактами и связь между элементами UI и моделью данных.
В этом классе реализованы обработчики сигналов, загрузка и сохранение данных, реакция на выбор элемента в списке контактов, а также оформление интерфейса через стили.

### ContactModel
Класс-модель, унаследованный от QStandardItemModel.
Отвечает за хранение данных о контактах в иерархическом виде.
Каждый контакт представлен как корневой элемент, а его поля (имя, фамилия, телефон, email, заметка) — как дочерние элементы.
Реализованы методы загрузки и сохранения данных в формате XML, чтобы сохранять все контакты на диск и восстанавливать их при запуске приложения.

### AddContactDialog
Диалоговое окно для добавления нового контакта.
Пользователь вводит данные (имя, фамилию, телефон, email и заметку), после чего по нажатию кнопки «Сохранить» данные передаются в основное окно и добавляются в модель.

Приложение представляет собой простую адресную книгу, которая позволяет пользователю хранить и редактировать список контактов. При запуске программы автоматически загружается список контактов из XML-файла. В левой части окна отображается список всех контактов, а в правой показываются подробные сведения о выбранном элементе. Когда пользователь выбирает контакт, его данные автоматически подставляются в соответствующие поля редактирования. Если выделение снимается, все поля справа очищаются и скрываются. Пользователь может изменять существующие контакты, добавлять новые с помощью отдельного диалогового окна или удалять ненужные записи. Все изменения сохраняются в XML-файл, чтобы при следующем запуске программы информация восстанавливалась автоматически. В меню «О программе» отображается окно с краткой информацией о версии приложения и его назначении. ПРиложение имеет аккуратный красивый интерфейс.
